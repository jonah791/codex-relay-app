### v1.0.5

- **合并启动/停止为模式切换**：右键菜单 → "使用 Relay" / "使用 OpenAI 默认"，一键切换并自动重启 Codex
- 模式指示器：当前模式前显示 ✓
- **事件驱动菜单**：每次右击托盘动态重建，彻底解决停止 Relay 后按钮不同步问题
- 退出时强制关闭所有连接（`closeAllConnections`）
- NSIS 安装包（含安装向导、桌面快捷方式、开始菜单）

### 使用

```
右键托盘图标:
  [供应商] ▸
  ──────────
  ✓ 使用 Relay           ← 一键启动 + 切换配置 + 打开 Codex
    使用 OpenAI 默认       ← 一键停止 + 恢复配置 + 打开 Codex
  ──────────
  重启 Codex
  设置...
  退出
```
